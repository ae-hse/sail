После успешного создания проекта Вы окажитесь на его главной странице. Здесь отображается основная информация по проекту, которую Вы ввели ранее. Нажав на Edit details, можно ее модифицировать. Справа отображены действующие участники проекта, чтобы добавить нового, необходимо нажать Add new member и ввести имя зарегистрированного пользователя, он будет добавлен к проекту автоматически. Внизу находятся дискуссионные топики по этому проекту, в которых Вы со своими единомышленниками можете вести обсуждение проекта. Пока там пусто, но Вы в любой момент можете начать новую тему, перейдя по ссылке Start one. Основу любого проекта составляет его база знаний. Перейдем к базе, нажав по ссылке go.
