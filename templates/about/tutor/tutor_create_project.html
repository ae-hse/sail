Продолжим ознакомление с процессом создания Вашего первого проекта. Чтобы создать проект, перейдите по вкладке Projects в верхнем правом углу. Заполните следующие поля:

- Name
Имя вашего проекта
- Slug
Сокращенное имя, присваиваемое системой Вашему проекту
- Description
Описание Вашего проекта

Нажмите Create, чтобы завершить создание нового проекта

Если Вы захотите изменить введенные данные, Вы всегда сможете вернуться и исправить эти поля. Чтобы показать возможности системы, мы будем рассматривать проект, исследующий зависимости в признаках стран европейского континента.
